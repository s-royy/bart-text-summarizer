# Text Summarization on Google Colab

A simple text summarization tool using Hugging Face Transformers v5+ on Google Colab.

## Features

✅ Handles long texts with automatic chunking  
✅ Free GPU acceleration on Google Colab  
✅ Transformers v5+ compatible  
✅ Interactive user input  
✅ High-quality BART-based summaries  

## Supported Models

- **facebook/bart-large-cnn** - General text and articles
- **google/pegasus-arxiv** - Scientific papers
- **t5-small** - Lightweight alternative

## Requirements

- Transformers (v5.0+)
- PyTorch

## Use Cases

- Article summarization
- Research paper abstraction
- Document condensation
- Content review
